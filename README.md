## What it does

StockOracle lets you type any stock ticker (AAPL, TSLA, MSFT etc.) and get a predicted next-day closing price powered by a Random Forest machine learning model trained on 2 years of historical data.

- Fetches real stock data via **yfinance**
- Trains a **Random Forest Regressor** on historical OHLCV data + engineered features
- Serves predictions through a **FastAPI** REST backend
- Displays results in a clean **HTML/CSS/JS** frontend
- Caches both data and trained models so repeat predictions are instant

## How it Works

```
User enters ticker (e.g. AAPL)
        ↓
FastAPI receives POST /predict request
        ↓
fetch_data.py → check cache → download if needed
        ↓
train_model.py → check saved model → train if needed
        ↓
predict.py → feed latest data into model → get prediction
        ↓
Frontend displays: current price, predicted price, change %
```

### Features used by the model
- Open, High, Low, Close, Volume
- 1-day, 5-day, 10-day price returns
- Moving averages (MA 10, 20, 50)
- Price vs moving average ratios
- 10-day volatility
- Volume change

## Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/stock-predictor.git
cd stock-predictor
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Start the API
```bash
uvicorn src.api.main:app --reload
```

### 4. Open the frontend
Open `src/ui/index.html` in your browser.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Check API is running |
| GET | `/health` | Health check |
| POST | `/predict` | Get price prediction for a ticker |
| GET | `/models` | List all cached trained models |

### Example request
```json
POST /predict
{
  "ticker": "AAPL",
  "force_retrain": false,
  "force_refresh": false
}
```

### Example response
```json
{
  "ticker": "AAPL",
  "current_price": 189.50,
  "predicted_price": 191.20,
  "change": 1.70,
  "change_pct": 0.90,
  "direction": "UP",
  "status": "loaded_from_cache"
}
```
